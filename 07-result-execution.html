结果执行与导出

从整理结果到可用知识：多格式导出与回写策略

版本: v1.0 | 更新日期: 2026-04-10 | 状态: 已完成
---

一、文档概述

1.1 背景

整理结果需要以用户可用的形式输出。支持多种导出格式，满足不同知识管理工具的需求，同时提供回写建议帮助用户优化原平台的收藏。

1.2 目标

  • • 支持4种主流导出格式
  • • 实现导出引擎的模块化设计
  • • 提供间接回写策略
  • • 设计导出中心UI
---

二、导出格式设计

2.1 格式概览

支持4种导出格式，满足不同场景的知识管理需求。

Markdown

适用于 Notion、语雀、Obsidian

PDF

适用于打印、归档、分享

思维导图

适用于 XMind、幕布

知识卡片

适用于 Anki、Quizlet

2.2 格式对比

格式 适用工具 特点 优先级
Markdown Notion, Obsidian 通用、易编辑 P0
PDF 通用 格式固定、易分享 P0
思维导图 XMind, 幕布 可视化结构 P1
知识卡片 Anki 适合记忆复习 P1

三、导出引擎实现

3.1 核心实现

// 导出引擎
class ExportEngine {
  constructor() {
    this.exporters = {
      markdown: new MarkdownExporter(),
      pdf: new PDFExporter(),
      mindmap: new MindMapExporter(),
      cards: new CardExporter()
    };
  }

  // 主导出方法
  async export(organizedData, format, options = {}) {
    const exporter = this.exporters[format];
    if (!exporter) {
      throw new Error(`不支持的导出格式: ${format}`);
    }

    // 预处理数据
    const processedData = await this.preprocessData(organizedData, format);
    
    // 执行导出
    const result = await exporter.export(processedData, options);
    
    return {
      format,
      content: result.content,
      filename: this.generateFilename(organizedData, format),
      size: result.content.length
    };
  }

  // 批量导出
  async exportBatch(organizedDataList, format, options = {}) {
    const results = await Promise.all(
      organizedDataList.map(data => this.export(data, format, options))
    );
    
    // 如果是多个文件，打包为zip
    if (results.length > 1) {
      return this.createZipArchive(results);
    }
    
    return results[0];
  }
}

四、间接回写策略

4.1 技术限制

限制说明

经调研，豆包、Kimi、通义千问、DeepSeek 均未开放修改用户对话收藏的API。因此无法实现自动回写，只能提供间接方案。

4.2 回写建议生成

// 间接回写建议生成器
class WritebackSuggestionGenerator {
  async generateSuggestions(organizedData, targetPlatform) {
    const suggestions = [];
    
    // 1. 标签优化建议
    organizedData.dialogs.forEach(dialog => {
      if (dialog.originalTags && dialog.optimizedTags) {
        const added = dialog.optimizedTags.filter(t => !dialog.originalTags.includes(t));
        const removed = dialog.originalTags.filter(t => !dialog.optimizedTags.includes(t));
        
        if (added.length > 0 || removed.length > 0) {
          suggestions.push({
            type: 'TAG_OPTIMIZATION',
            dialogId: dialog.id,
            title: dialog.summary,
            action: 'UPDATE_TAGS',
            current: dialog.originalTags,
            suggested: dialog.optimizedTags,
            platform: targetPlatform
          });
        }
      }
    });
    
    return suggestions;
  }
}
